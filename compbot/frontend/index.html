üöÄ Quick Setup

1

Add TournaBot to Your Server

Use the invite link above with Administrator permissions.

2

Run Setup Command

In your Discord server, type: /setup

This will auto-create all necessary channels and roles.

3

Configure Channels (Optional)

If you want to use existing channels:

/channels set type:announcements
/channels set type:brackets
/channels set type:verifications
4

Create Your First Tournament

Use: /tournament create

All announcements will go to your configured channels automatically!
